# TOOLS

## 工具与能力定义

### executor.py

- SSH 执行
- 环境感知
- 风险识别
- 执行前状态检查
- 执行后状态验证
- 命令拦截

### modules/audit_store.py

- 审计日志持久化
- 导出
- 清空

### voice_input.py

- 语音转写
- 自动回填输入框

### modules/c_executor.py

- C 代码清洗
- 远端编译
- 运行结果回传

### modules/document_parser.py

- AI 主导文档内容抽取
- 规则兜底
- 命令识别
- C 代码识别

### modules/windows_terminal.py

- Windows 终端定位
- 执行结果镜像展示
