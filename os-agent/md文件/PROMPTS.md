# PROMPTS

## 核心系统 Prompt

项目中的核心 Prompt 位于 `agent.py`，作用是将自然语言转换为严格 JSON 命令计划。

关键要求包括：

- 优先理解上下文
- 输出严格 JSON
- 多步任务必须返回 `steps`
- 风险做初判，最终以本地规则为准
- 说明文字使用与用户一致的语言

## 语音转写 Prompt

项目中的语音转写 Prompt 位于 `voice_input.py`：

```text
请准确转写这段语音内容，只输出识别出的文本，不要补充说明。
```

## 反馈总结 Prompt

根据命令执行结果动态切换：

- 高风险拦截说明
- 中风险提醒
- 失败原因解释
- 正常结果摘要
