# 工具及能力定义

## 1. UI 与交互

- `app.py`
- 能力：聊天展示、风险标签、历史记录、语音录音、语音自动填入、复杂任务确认、审计日志持久化、环境信息展示与结果展示

## 2. Agent 决策

- `agent.py`
- 能力：意图解析、上下文利用、复杂任务拆解、自然语言反馈生成

## 3. SSH 执行

- `executor.py`
- 能力：远程命令执行、环境信息感知、命令安全检查、执行前状态检查、执行后状态验证、风险识别与拦截

## 4. 编程任务执行

- `modules/c_executor.py`
- 能力：生成 C 代码后的上传、编译、执行与回显

## 5. 本地终端桥接

- `modules/windows_terminal.py`
- 能力：在 Windows 环境中提供本地终端能力支持

## 6. 文档输入扩展

- `modules/document_parser.py`
- 能力：以 AI 为主、规则兜底，从文档中批量提取 Linux 命令和 C 代码并构造执行流程

## 7. 审计日志存储

- `modules/audit_store.py`
- 能力：审计日志落盘、读取、清空、导出文本

## 8. 语音输入能力

- `voice_input.py`
- 能力：
  - 接收录音文件字节流
  - 转换为 Base64 Data URL
  - 调用千问 ASR 模型识别
  - 返回纯文本识别结果
  - 配合前端脚本自动回填聊天输入框

## 9. 风险控制能力

- 普通用户创建/删除默认低风险
- 密码修改、权限修改、文件删除等识别为中风险
- 删除核心目录、格式化磁盘、远程脚本直连执行等识别为高风险
- 中高风险执行前要求确认或直接拒绝
