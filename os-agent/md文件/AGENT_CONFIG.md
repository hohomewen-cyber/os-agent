# AGENT CONFIG

## 模型配置

- `QWEN_API_BASE`：千问兼容模式 API 地址
- `QWEN_API_KEY`：鉴权密钥
- `QWEN_MODEL`：主对话模型
- `QWEN_ASR_MODEL`：语音识别模型

## Agent 运行职责

- 接收自然语言输入
- 结合上下文生成命令计划
- 将复杂任务拆成多步执行
- 生成结果反馈说明
- 结合远程环境信息调整命令生成

## 决策边界

- 命令是否真正执行，由 `executor.py` 的风控规则最终裁定
- Agent 负责理解与规划，执行器负责环境感知、安全兜底和状态验证

## 会话能力

- 保存有限历史上下文
- 识别语言偏好并支持在对话中切换
- 支持多轮连续任务
